## Getting Started

Prerequisites

- [nodejs](https://nodejs.org/en/)
- [git](https://git-scm.com/)

## Installation

Clone the project

```bash
  git clone https://github.com/renatoallart/portfolio.git
```

Go to the project directory

```bash
  cd portfolio
```

Install dependencies

```bash
  npm install
  #or
  yarn add
```

Start the application

```bash
  npm run dev
  #or
  yarn dev
```
